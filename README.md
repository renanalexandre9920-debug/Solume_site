# SoLume — Site estático (Go Live)

✅ Feito para rodar no VS Code com **Live Server / Go Live**.

## Rodar
- Abra a pasta no VS Code
- Clique com o botão direito em `index.html` → **Open with Live Server**

## Personalizar WhatsApp e Certificações
Edite: `assets/js/data.js`
- `WHATSAPP_NUMBER`
- `CERTIFICATIONS`

## Trocar fotos (profissional e família)
Coloque seus arquivos em `assets/img/` e troque no `index.html`:
- `assets/img/foto_placeholder.svg` → `assets/img/foto.jpg` (ou .png)
- `assets/img/familia_placeholder.svg` → `assets/img/familia.jpg` (ou .png)

Dica: use fotos claras, sorrindo e com fundo neutro.
